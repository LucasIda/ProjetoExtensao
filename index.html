Nossa proposta:

A proposta do nosso projeto é desenvolver um protótipo, onde através dele será possível verificar os seguintes dados do solo: umidade, temperatura e pH. Com estes dados coletados, será possível realizar comparações com a situação que é considerado saudável para o solo, e então propor soluções para manter o solo em seu estado recomendado para o crescimento saudável do cultivo.
